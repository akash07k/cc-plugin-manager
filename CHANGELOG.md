# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/),
and this project adheres to [Semantic Versioning](https://semver.org/).

This file is regenerated automatically by [`git-cliff`](https://git-cliff.org/)
from Conventional Commits whenever a `v*` tag is pushed -- see
[`.github/workflows/release.yml`](.github/workflows/release.yml). Manual edits
will be overwritten on the next release.
